# Media Kit | Natalia Gilles

Media Kit desenvolvido para apresentar minha presença digital, perfil de audiência, resultados, feedbacks e oportunidades de colaboração com marcas.

## Sobre

Este projeto reúne informações sobre:

* Posicionamento de marca pessoal
* Redes sociais e métricas de alcance
* Feedbacks da audiência
* Estudo de público
* Marcas parceiras
* Informações para contato e colaborações

## Tecnologias Utilizadas

* HTML5
* CSS3
* Design Responsivo
* Git & GitHub

Apresentar de forma profissional meu trabalho como criadora de conteúdo, destacando alcance, engajamento e oportunidades de parceria para marcas e empresas.
